# CargoClass

Landing page for **CargoClass** — solar-powered, offline-AI microschools built inside shipping containers and delivered anywhere.

🌐 **Live site:** https://minecraftpro546.github.io/cargoclass/

## About

A single-page marketing site presenting the CargoClass concept: the problem of education inequality, how a container-based school works, what's inside the pod, who it reaches, and the path from prototype to community. It features a scroll-driven neon line that draws down the page as you read, animated reveals, and count-up stats.

## Structure

- `index.html` — the entire site (self-contained: HTML, CSS, and JS in one file).

## Develop

It's a static page with no build step. Open `index.html` directly in a browser:

```
start index.html      # Windows
```

Edit `index.html` and refresh to see changes.

## Deploy

The site is hosted on GitHub Pages from the `main` branch root. Pushing to `main` redeploys automatically (~1 minute):

```
git add index.html
git commit -m "your message"
git push
```
